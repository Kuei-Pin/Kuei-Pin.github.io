异步读写文件NSStream

发表于 2018-05-05 | | 阅读次数:
字数统计: 367

NSStream是一种异步读写文件的技术，当你需要从硬盘或者Socket链接中将以字节流的形式读取数据，或者将文件以字节形式写入硬盘或者以Socket形式发送给远程服务器时，可以使用。

使用

当我们从硬盘中读取文件到内存时，一般更多的是通过NSData来读取。在读取小文件时，直接使用NSData不会有什么影响，但是如果读取的文件大小比较大时，使用NSData直接读取文件，可能会阻塞当前进程，测试代码如下：
